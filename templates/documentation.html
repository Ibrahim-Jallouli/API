Documentation API Image Processing

/upload (POST)

Endpoint for uploading an image.

Usage:

POST http://127.0.0.1/upload

/invert_horizontal (POST)

Endpoint for horizontally inverting an image.

Usage:

POST http://127.0.0.1/invert_horizontal

/invert_vertical (POST)

Endpoint for vertically inverting an image.

Usage:

POST http://127.0.0.1/invert_vertical

/invert_colors (POST)

Endpoint for inverting the colors of an image.

Usage:

POST http://127.0.0.1/invert_colors

/rotate (POST)

Endpoint for rotating an image by 90 degrees.

Usage:

POST http://127.0.0.1/rotate

/change_color (POST)

Endpoint for converting an image to black and white.

Usage:

POST http://127.0.0.1/change_color

/download_image (GET)

Endpoint for downloading the transformed image.

Usage:

GET http://127.0.0.1/download_image
